# Lab 3: Sensor Data Visualization

This project generates simulated temperature readings from two sensors and visualizes them using scatter plots, histograms, and box plots. The notebook also displays a combined PNG image created by a Python script.

## Installation
1. Install Python 3.x  
2. Install required packages:

pip install matplotlib numpy pillow


## Usage
Run the script to generate the combined plot:
python generate_plots.py


Open the Jupyter notebook to view the plots and analysis:
lab3_sensor_plots.ipynb
